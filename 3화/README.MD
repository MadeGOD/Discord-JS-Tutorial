### 잠시 튜토리얼을 보시기 전 읽어주세요.

이번 3화 튜토리얼은 커맨드 핸들링에 관련하여 제작한 겁니다.   
소스 코드를 사용하시는 것은 좋으시나 꼭 소스를 이해하셨으면 바램을 가져봅니다...   
봇 소스만 복붙하여 사용하는 행위는 절대로 좋은 행위가 아닙니다.   
튜토리얼을 제작하는 목적은 봇을 입문하시는 분들을 위하여 공부하시라고 제작하는거지 복붙 하라고 만든 용도가 아닙니다..
